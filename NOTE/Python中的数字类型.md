### Python中的数字类型

1. **int（整数）**：整数类型用于表示整数，**不带小数点**

示例：


```python
x = 10
print(type(x)) 
```
2. **float（浮点数）**：浮点数类型用于表示**带有小数点**的数字。

示例：


```python
y = 3.14
print(type(y))  
```
3. **bool（布尔）**：布尔类型有两个值，**True**和**False**。

示例：


```python
z = True
print(type(z)) 
```
4. **complex（复数）**：复数类型用于表示包含**实部**和**虚部**的数字。

示例：


```python
a = 3 + 4j
print(type(a))  
```
